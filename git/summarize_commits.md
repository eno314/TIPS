# コミットをまとめる

## 手順

1. `rebase -i HEAD~[まとめたいコミット数]` もしくは `rebase -i [まとめたいコミットの直前のコミット番号]` を実行する
1. 最初の行はそのまま、2行目以降は `pick` の部分を `s` か `f` に置き換える
1. `s` を選択した場合は、コメントを編集する
1. まとめたいコミットを push してしまっていた場合は、 `git push -f origin [ブランチ]`
